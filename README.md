# ☕ Coffee Shop Website

A modern full-stack Coffee Shop web application built with React, Node.js, and MongoDB.

---

## 🚀 Features

- 🏠 Beautiful Home Page (Hero + Featured Coffee)
- 📋 Coffee Menu
- 📝 Blog / Customer Reviews
- 🔐 User Authentication (Login/Register)
- 👨‍💼 Admin Dashboard (Manage products & users)
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- DaisyUI

### Backend (planned)
- Node.js
- Express.js

### Database (planned)
- MongoDB

## ⚙️ Installation

Clone the repository:

```

git clone https://github.com/yourusername/coffee-shop.git

```

Go to project folder:

```

cd coffee-shop

```

Install dependencies:

```

npm install

```

Run the project:

```

npm run dev

```
